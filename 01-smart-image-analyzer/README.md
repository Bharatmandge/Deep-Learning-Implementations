# Smart Image Analyzer

A mini computer vision pipeline that combines:

- Image Classification (ResNet)
- Object Detection (YOLOv8)
- Image Enhancement (OpenCV)

## Features
- Classifies input image
- Detects objects with bounding boxes
- Enhances image using sharpening filter

## How to Run

pip install -r requirements.txt
python main.py
