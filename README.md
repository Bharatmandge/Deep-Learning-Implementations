# 🧠 Deep Learning Implementations

A structured collection of hands-on deep learning projects built to move from **theory → real implementation**.

Instead of just studying concepts, this repository focuses on actually building and understanding how they work in practice.

---

## 🚀 Projects

### 📌 1. Smart Image Analyzer
- Image Classification (CNN / Transfer Learning)
- Object Detection (YOLO)
- Image Enhancement (OpenCV)

👉 Combines multiple basic vision tasks into a single pipeline

---

### 📌 2. Multi-Task Vision Pipeline
- Object Detection (YOLOv8)
- Image Segmentation (U-Net)
- Evaluation Metrics (IoU, Dice)
- End-to-end pipeline: Detect → Crop → Segment → Evaluate → Visualize

👉 Simulates how real-world computer vision systems are built

---

## 🧠 Key Concepts Covered

- Convolutional Neural Networks (CNNs)
- Transfer Learning
- Object Detection
- Image Segmentation (Encoder–Decoder, U-Net)
- Performance Metrics (IoU, Dice Score)
- Data Flow between OpenCV, NumPy, and PyTorch
- Modular Code Design
